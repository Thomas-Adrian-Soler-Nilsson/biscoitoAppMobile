## ✨ Funcionalidades

✅ Exibe frases aleatórias de “biscoito da sorte”.  
✅ Interface simples, moderna e intuitiva.  
✅ Atualização instantânea da frase ao clicar no botão.  
✅ Desenvolvido com **React Native** e **Expo**, compatível com Android e iOS.
